## Install
To use this template, simply follow these steps
### Step 1
Install Laravel dependencies
```
composer install
```

### Step 2
Install packages
```
npm install
```

### Step 3
```
 npm run dev
```

### Step 4
```
php artisan migrate
```
### Done! :)!!
